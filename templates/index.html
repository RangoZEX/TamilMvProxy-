How this app uses the data using OAuth?

This is a Telegram bot that let users connect their account to the Bot.
This bot basically helps the users to manage there google drive account through a telegram bot. It simplifies the work of user in uploading files through a direct link and also helps in managing there accounts. The client id and client secret will generate a authentication url and sent to users via a telegram bot message. The authentication url is given in Telegram bot itself only when user requests. The user visit the auth link and login using their gmail account and sends back the success code. Then the access token is generated for the user. Bot saves the access token in a secure mongo database for later use for the user.
we also provide a /logout command in our bot which helps the user to logout from there account when ever the want and we delete there information from our database when the use logout commad.

Note: Access token is used only when user send any request to the bot and it cannot be misused

How this app enhances user functionality?

Users need an easy client to upload their files to there drive links to upload to Google Drive. Downloading files is sometimes slow. Then it will be very easy if there is an app that downloads the files for them and upload to their Google Drive. Google Drive can give high speed downloads, so users always prefers to upload files to Google Drive using url and download from it. This app just make their work easy by just sending their direct link to Telegram bot and uploading it to their drive.it will be there in the drive and whenever they want they can Download at high speed.
